# Create a polished README via btw's guided task (reuse)

Create a polished README via btw's guided task (reuse)

## Usage

``` r
codeagent_create_readme(client = NULL, mode = "console", ...)
```

## Arguments

  - client:
    
    A codeagent client or an `ellmer::Chat` (its chat is reused).

  - mode:
    
    One of `"console"`, `"app"`, `"client"`, `"tool"`.

  - ...:
    
    Passed to `btw::btw_task_create_readme()`.

## Value

See `btw::btw_task_create_readme()`.
