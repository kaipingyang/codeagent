# Register the plan-mode tools on a chat

Register the plan-mode tools on a chat

## Usage

``` r
register_plan_mode_tools(chat, mode_env)
```

## Arguments

- chat:

  An [`ellmer::Chat`](https://ellmer.tidyverse.org/reference/Chat.html)
  object.

- mode_env:

  Environment with a live `$mode` slot.

## Value

Invisibly `chat`.
