# Register the TodoWrite tool on a chat

Register the TodoWrite tool on a chat

## Usage

``` r
register_todo_tool(chat, session_id = "default")
```

## Arguments

- chat:

  An [`ellmer::Chat`](https://ellmer.tidyverse.org/reference/Chat.html)
  object.

- session_id:

  Character. Session id for the todo file.

## Value

Invisibly `chat`.
