# Write a memory to disk

Write a memory to disk

## Usage

``` r
write_memory(title, content, description = "")
```

## Arguments

  - title:
    
    Character. Short human title (also the filename slug).

  - content:
    
    Character. The fact to remember.

  - description:
    
    Character. One-line summary for the index/recall.

## Value

Invisibly, the file path written.
