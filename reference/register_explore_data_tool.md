# Register the ExploreData tool on a Chat

Register the ExploreData tool on a Chat

## Usage

``` r
register_explore_data_tool(chat, envir = .GlobalEnv)
```

## Arguments

- chat:

  An [`ellmer::Chat`](https://ellmer.tidyverse.org/reference/Chat.html)
  object.

- envir:

  Environment from which to read data.frames.

## Value

Invisibly `chat`.
