# Create a bare ellmer Chat from settings

Create a bare ellmer Chat from settings

## Usage

``` r
.make_chat(settings, cwd = getwd(), ...)
```

## Arguments

  - settings:
    
    List. Output of `load_settings()`.

  - cwd:
    
    Character. Working directory.

  - ...:
    
    Passed to the underlying ellmer function.

## Value

An `ellmer::Chat` object.
