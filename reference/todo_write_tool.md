# Create the TodoWrite tool

Create the TodoWrite tool

## Usage

``` r
todo_write_tool(session_id = "default")
```

## Arguments

- session_id:

  Character. Session id used to name the todo file. The harness passes
  the live session id; defaults to `"default"`.

## Value

An [`ellmer::tool()`](https://ellmer.tidyverse.org/reference/tool.html)
object.
