# Register the TeamRun tool on a chat

Register the TeamRun tool on a chat

## Usage

``` r
register_team_tool(chat, model = NULL, cwd = getwd())
```

## Arguments

  - chat:
    
    An `ellmer::Chat` object.

  - model:
    
    Character. Default model for team agents.

  - cwd:
    
    Character. Working directory.

## Value

Invisibly `chat`.
