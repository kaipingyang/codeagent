# Create the Edit tool

Create the Edit tool

## Usage

``` r
edit_tool(mode = "default", rules = list(), ask_fn = NULL)
```

## Arguments

  - mode:
    
    Character. Permission mode.

  - rules:
    
    List. Permission rules.

  - ask\_fn:
    
    Function or NULL.

## Value

An `ellmer::tool()` object.
