# Create the ExploreData tool

Create the ExploreData tool

## Usage

``` r
explore_data_tool(envir = .GlobalEnv)
```

## Arguments

- envir:

  Environment from which to read data.frames. Defaults to `.GlobalEnv`.

## Value

An [`ellmer::tool()`](https://ellmer.tidyverse.org/reference/tool.html)
object.
