Export this exploration session to a reproducible Quarto document.

Call the `GenerateReport` tool with an optional title.

The generated `.qmd` file will contain:
- Each question asked during the session as a section heading
- The R/dplyr code generated by the agent as executable code blocks
- The results and analysis as text
- All in a reproducible format that can be rendered with `quarto::quarto_render()`
