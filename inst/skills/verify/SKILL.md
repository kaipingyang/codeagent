Please verify the last action or implementation using this structured checklist:

1. **Correctness** -- Does the code/output do what was intended?
2. **Completeness** -- Are all required parts present?
3. **Edge cases** -- Have obvious edge cases been handled?
4. **Tests** -- Do existing tests still pass? Are new tests needed?
5. **Side effects** -- Are there any unintended side effects?

$ARGUMENTS

Provide a concise pass/fail assessment for each item and a summary verdict.
